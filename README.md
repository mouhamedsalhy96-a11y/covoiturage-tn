# CovoiturageTN

CovoiturageTN is a full-featured ridesharing web application demonstrating end-to-end product workflows, including user onboarding, trip management, dashboards, messaging, and admin reconciliation tools.

## Overview

This project simulates a real-world ridesharing platform with clearly defined user roles and realistic business logic.

Users can:
- Register and authenticate
- Search available trips
- View detailed trip information
- Book seats and manage reservations
- Message drivers and other participants
- Manage activity from a personal dashboard

Drivers can:
- Publish new trips
- Track reservations and messages
- Manage trip availability

Administrators can:
- Review bookings
- Reconcile cash-on-delivery (COD) payments
- Monitor payment status and booking progress

## Features

- User authentication and protected routes
- Trip search and listing system
- Detailed trip pages with route timeline and map
- User dashboard for bookings, messages, and trips
- Admin dashboard for cash-on-delivery reconciliation
- Localisation and multi-language support
- Role-based views (passenger, driver, admin)

## Tech Stack

- Framework: Next.js
- Database & ORM: Prisma
- UI: Component-based responsive design
- Maps: Route visualisation
- Deployment: Vercel

## Development Notes

This project was built using an AI-assisted development workflow to accelerate UI iteration, structure, and feature development, with manual review and refinement throughout to ensure correctness and maintainability.
